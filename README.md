# Github-IFTTT-WeChat

## 动机

[IFTTT](https://ifttt.com) 中的 That 暂时没有打通微信的服务，所以想借助 Server 酱完成这个动作。

订阅一些仓库的 release 然后通过 Server 酱发送到你的个人微信。

IF THIS(React released) THEN THAT(send WeChat message to me).
